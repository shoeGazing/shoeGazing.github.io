首次在github上创建repository

> cd `the directory of the repo`
>
> git remote set-url origin https://github.com/usrname/repo_name
> 
> git pull origin master
> 
> git push -u origin master

md 转成html
> jekyll build

向repo实现推送

> git add -A
>
> git commit -m "okay"
>
> git push -u origin master
